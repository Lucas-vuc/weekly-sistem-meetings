What This Replaces

×
Jira
No more double-entry. The meeting writes directly to the database. Source of truth lives where work is decided.
×
JSON sneakernet
No export → DM → import. All meeting tools read and write the same shared databases in real time.
×
Future branch (paused)
Ideas / Pipeline / Incubation archived. Data preserved, no active triage until reactivated.

What This Adds (vs. previous system)

+
Real-time multi-user editing. Founders see changes instantly without refresh or import.
+
Permanent Portfolio Matrix view. Decision Engine scoring (6 Quick Kill filters + 8 dimensions) accessible any time, not only during the Bi-monthly.
+
Top 5 Live. Scores editable continuously with 2-founder ratification and full audit log.
+
KPI Dashboard live. Target vs actual, trend, alerts. Auto-feeds Monthly Part 1.
+
Linked objectives. Weekly tasks → Monthly Objective → Top 5 Project. Progress rolls up automatically.
+
Mobile + offline. Notion native apps for iOS/Android. Critical for Emmanuel on-site in Ghana.

Stage 1 — Quick Kill Filter

Six binary gates. A project must pass all six to enter scoring. Any "No" sends the project to DISREGARD until the gate is resolved.

Gate 1
Capital available?
Gate 2
Payback ≤ 48 mo?
Gate 3
IRR ≥ 15%?
Gate 4
No fatal risk?
Gate 5
Team capacity?
Gate 6
Strategic fit?
